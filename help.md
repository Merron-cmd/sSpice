# 电路模拟器 - 使用说明

## 启动程序
运行可执行文件后，显示主菜单：
```
=== Main Menu ===
1. Read netlist from file
2. Exit
Choose:
```
- 输入 `1` 读取网表文件，按提示输入文件名（如 `RC.net`）。
- 输入 `2` 退出程序。

## 读取网表后的子菜单
成功读取网表后，进入电路菜单：
```
--- Circuit Menu ---
1. DC analysis
2. AC single frequency analysis
3. AC sweep analysis
4. Return to main menu
Choose:
```

### 1. 直流分析（DC）
- 选择 `1`，程序自动计算各节点直流电压。
- 结果同时显示在屏幕并写入 `<网表名>DC.simout` 文件。

### 2. 交流单频率分析（AC Single）
- 选择 `2`，提示 `Enter frequency (Hz):`，输入频率（可带单位，如 `1k` 表示 1000 Hz）。
- 程序计算该频率下所有节点的电压（实部/虚部）。
- 屏幕显示带表头，同时写入 `<网表名>AC.simout`（第一行为频率，后续为数据，无表头）。

### 3. 交流扫频分析（AC Sweep）
- 选择 `3`，依次提示输入：
  - **主动源名称**：如 `V1`（必须是网表中定义的 AC 源，程序会验证）。
  - **幅值**：可带单位（如 `1m`）。
  - **起始频率**和**终止频率**：可带单位（如 `10` 和 `1M`）。
  - **扫频类型**：`1` 线性，`2` 十倍频（Decade）。
  - **点数**：
    - 线性：总点数（含端点）。
    - 十倍频：每十倍频的间隔数（总点数为 `间隔数 × 十倍频数 + 1`）。
- 结果仅写入 `<网表名>ACsweep.simout`，包含表头 `Freq(Hz)` 及每个节点的 `_real` 和 `_imag` 列。

## 网表文件格式
- 第一行：节点总数（节点编号从 `0` 开始，`0` 为地）。
- 后续每行：`<元件名> <正节点> <负节点> <参数>`
  - **元件名**首字母区分类型（不区分大小写）：
    - `R/r`：电阻，参数为阻值。
    - `C/c`：电容，参数为容值。
    - `L/l`：电感，参数为感值。
    - `V/v`：电压源，参数可为 `DC <值>` 或 `AC <幅度> <频率> <相位(度)>`，省略 `DC` 时默认为直流。
    - `I/i`：电流源，格式同电压源。
  - **参数支持工程前缀**：`u`/`U`（微）、`m`（毫）、`k`/`K`（千）、`M`（兆）、`G`（吉）、`T`（太）。

对于电源，第一个节点为正，第二个为负。

### 网表示例（一阶 RC 低通）
```
2
V1 1 0 AC 1 100 0
R1 1 2 1k
C1 2 0 1u
```

## 输出数值精度
- 直流电压：固定 6 位小数。
- 交流电压实部/虚部：绝对值 < 0.001 时使用科学计数法（8 位有效数字），否则固定 8 位小数，保证小数值可见。

## 注意事项
- 电路必须至少有一个元件连接到节点 `0`（地），否则可能矩阵奇异。
- 扫频时主动源名称需与网表完全一致（大小写敏感）。
- 频率和幅值输入均支持工程前缀。
